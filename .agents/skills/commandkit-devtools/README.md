# commandkit-devtools skill

Skill for local development debugging using `@commandkit/devtools`.

- enabling devtools in a CommandKit bot
- troubleshooting command/event behavior in development
- validating local diagnostics setup

- local dev environment details
- symptoms/bugs seen during runtime
- current plugin configuration

## Expected outputs

- correct plugin wiring
- diagnosis-oriented adjustments
- practical local verification steps
