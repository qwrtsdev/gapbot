# commandkit-queue skill

Skill for queue architecture with `@commandkit/queue`.

## Use this skill when

- implementing publish/subscribe bot workflows
- configuring queue drivers (e.g., Redis-backed)
- decoupling message processing from command execution

## Typical inputs

- producer/consumer flow
- transport/driver constraints
- delivery reliability requirements

## Expected outputs

- driver setup
- channel/message contract strategy
- robust send/receive integration
